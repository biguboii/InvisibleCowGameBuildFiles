3D INVISIBLE COW FINDER


Team Name: GenGame Productions

Members: Jacob Romio, AJ Ricafrente, Adam Hancock, Victor Tran

Description: "Puzzle" game where the player must find an invisible cow based on the audio it emits.

How to play: Use wasd or arrow keys to move while using the mouse to look around, left-shift to sprint, and space to jump. Move towards the audio that the cow makes to run into it and win the game.

Citations: http://soundbible.com/1572-Single-Cow.html
